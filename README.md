# proxy-switch-app

This is a simple proxy switcher inspired by [SwitchyOmega](https://github.com/FelisCatus/SwitchyOmega).

## Features

Proxy backend:

- [ ] http/https
- [ ] socks

Operation system:

- [ ] macos
- [ ] windows
- [ ] linux

## Getting started

```
cargo install tauri-cli --version "^2.0.0"
npm install
cargo tauri dev
```
